# The Hearth

Nick Ngoh's public portfolio site.

## Stack

- Vite
- React
- TypeScript
- Framer Motion
- Railway-ready static deployment with `serve`

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deploy

Railway will use `railway.json`:

- Build: `npm run build`
- Start: `npm run start`

For a pure static portfolio, Vercel or Netlify may be simpler than Railway. Railway is still fine if you want to keep your projects on one deployment platform.
